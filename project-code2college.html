Learning to Think in Code

Code2College is where my programming skills crossed from hobbyist to professional. Over nine weeks, I moved through foundational and advanced Python concepts, built multiple real projects, and gained the professional development skills to present technical work confidently.

As a mechanical engineer, I already understood systematic problem-solving — breaking a complex system into components, testing each one, and integrating back. Code2College showed me that software engineering follows the same logic. Python became another tool in my engineering toolkit, not a separate discipline.

Beyond the technical curriculum, the program's professional development workshops covered time management, workplace communication, and portfolio strategy — skills that directly translate to engineering internships and research environments.

Key Contributions

  • Completed a 9-week web programming course covering foundational through advanced Python concepts including data structures, functions, file I/O, and object-oriented programming.
  • Developed multiple technical projects in Python applying intermediate concepts — including data manipulation, automation scripts, and logic-based applications.
  • Attended professional development workshops focused on time management, technical communication, and portfolio development for industry readiness.
  • Applied programming skills to engineering contexts — bridging the gap between software development and mechanical systems design.
